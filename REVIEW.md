# Review Path

Start with `index.html` and open the Operational Surface.

Use the visible sequence:

```text
stage movement
resolve material standing
emit witness
bind reviewed standing
introduce material drift
```

The surface demonstrates that a reviewed standing may become stale when the active basis changes.

For executable proof, run:

```bash
python3 examples/negative_cases.py
python3 examples/operational_lifecycle.py
```
